# E.C.H.O Monitoring

Dashboard web statis untuk monitoring sistem E.C.H.O.

## Fokus tampilan

- Monitoring live eye tracking.
- Status pasien, arah pandangan, confidence, blink rate, dan latency.
- Output pesan pasien non-verbal.
- Status kamera, unit proses, layar, dan speaker.
- Alert prioritas dan activity log.

## Cara membuka

Buka `index.html` langsung di browser. Tidak perlu instal dependensi.
